About This App

This is a full-stack demo application built using Angular for the frontend, Express.js for the backend, and MongoDB for the database.

The app allows users to add, view, update, and delete products in a simple and intuitive interface. It's designed to demonstrate how modern web technologies can work together to build scalable and maintainable applications.

Whether you're learning the MEAN stack or building your own inventory system, this app is a great starting point.
